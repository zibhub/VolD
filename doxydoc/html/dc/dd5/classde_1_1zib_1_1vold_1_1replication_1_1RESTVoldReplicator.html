Detailed Description

Replicator delegating all requests to another vold REST based service.

Note:
Since the vold frontend has another interface language (using the class Key for keys instead of lists of strings), the keys have to be build up. Hence, when using this replicator, the keys given here need the format specified for Key.buildkey.
