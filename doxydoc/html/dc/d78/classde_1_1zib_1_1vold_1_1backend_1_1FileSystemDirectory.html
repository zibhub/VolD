Detailed Description

Implementation of PartitionedDirectoryBackend based on a file system structure.

Here, a key will be a subdirectory and the values will be its files. To avoid name clashing of keys and files, a plus ('+') will be prepended to keys (i.e. to directory names) and a minus ('-') will be prepended to files (i.e. to values). Different partitions will be represented by different directories in the root.

See also:
PartitionedDirectoryBackend
Author:
Jörg Bachmann (bachmann@zib.de)
