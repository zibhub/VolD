Detailed Description

Interface for Backends with slice information.

For each entry the timeslice of the time of the insertion will be remembered, thus the map of key-valuelists can be returned within a given timeslice.

Each slice has the same size (given in seconds). The slice room is a factor ring. The actual slice number is calculated by the actual time by dividing it by timeslicesize and taking the first representative in the ring mod numberofslices. Hence, the slice is a number between 0 and numberofslices-1.
