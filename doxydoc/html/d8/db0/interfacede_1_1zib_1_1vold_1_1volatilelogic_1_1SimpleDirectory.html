Detailed Description

Interface for volatile directory usage.

Each volatile directory must implement this interface. It provides the methods used for read and write access by the Frontend.

This interface is just one part of the main interface VolatileDirectory exported by this package.

Note:
The backend uses lists of strings for storing values. Since there are some backends having a set semantic nonetheless, the set semantic is enforced here.
